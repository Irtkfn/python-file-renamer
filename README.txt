# ✨ Возможности

- **🔢 Умная нумерация**: автоматически добавляет ведущие нули (001, 002... 100).
- **🛡️ Безопасность**: скрипт проверяет наличие файлов, чтобы ничего случайно не перезаписать.
- **📁 Сохранение расширений**: меняется только имя, формат файла (jpg, pdf, txt) остается прежним.
- **⚡ Лимит**: по умолчанию обрабатывает первые 100 файлов (легко изменить в коде).

---

# 📸 Как это выглядит

| Было (Old Name) | Стало (New Name) |
| :--- | :--- |
| `IMG_8821.jpg` | `vacation_001.jpg` |
| `data_final.csv` | `vacation_002.csv` |
| `notes.txt` | `vacation_003.txt` |

# 🚀 Быстрый старт

### 1. Подготовка
Убедитесь, что у вас установлен [Python](https://www.python.org/downloads/).

### 2. Установка
Склонируйте репозиторий или просто скачайте файл `main.py`:
```bash
git clone https://github.com/Irtkfn/python-file-renamer.git
cd python-file-renamer
```
### 3. Запуск
python main.py
